Loop Lab teaches the same engine the real Loop runs: plan → act → observe, a reflect back-edge, a thrash guard, and "you can't fake done." One concept per level — knowledge builds up as you go. Every block and every cycle costs loop-time, so each level has an effort budget: pick the right blocks, not all of them. Progress is saved in your browser.
