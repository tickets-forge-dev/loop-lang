Welcome to LoopFlow Studio

Learn the LoopFlow language by building it. Each chapter teaches one idea — snap blocks into a loop, hit Run, and watch it plan → act → observe. You'll build a real system, one loop at a time.

LoopFlow Lab teaches the same engine the real LoopFlow runs: plan → act → observe, a reflect back-edge, a thrash guard, and "you can't fake done." One concept per chapter — knowledge builds up as you go. Every block and every cycle costs loop-time, so each chapter has an effort budget: pick the right blocks, not all of them. Progress is saved in your browser.
