Keywords / plan from

plan from

Source the plan externally. Ops & reuse

Syntax

plan from archon

What it does

Instead of generating the plan, source it from an external system. plan from archon pulls the next task/plan from an Archon project; Loop drives Claude Code to execute it and writes results back, keeping the two in sync.

Example

loop "work the next Archon task":
  goal: implement the next planned task
  plan from archon
  each cycle: plan, then act, then observe
  done when "pnpm test" passes
external plan source
