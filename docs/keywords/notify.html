Keywords / notify

notify

Where to send completion. Ops & reuse

Syntax

notify: <destination>

What it does

Notification destination on completion (e.g. slack) — the config tier's "tell someone when it's done". Deferred past v1 for execution; parses today.

Example

schedule: nightly
notify: slack
ping on finish
